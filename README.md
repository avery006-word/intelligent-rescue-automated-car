# intelligent-rescue-automated-car
远程控制文件夹中的代码是基于Arduino板开发的，实现了PS2手柄与Arduino的通信，以及对电机的驱动，让PS2手柄可以远程控制小车进行移动、抓取和释放。

视觉识别文件夹中的代码实现摄像头视野范围内形状、颜色的识别，从而识别指定目标和围栏，并判断距离，确保小车自主抓取目标并在指定区域释放。

main1.0.py  

功能：主控流程，循环执行“抓球→送球”任务

首先调用 grab_first_ball.run() 抓第一个球，成功后调用 deliver_ball.run() 送球。

随后进入循环：调用 find_and_grab_ballqq.run() 抓球，再调用 deliver_ball.run() 送球，如此重复。

对每个子模块的返回值进行有效性检查，遇到异常则跳过当前循环继续执行。

grab_first_ball.py

功能：抓取第一个符合目标颜色的球

通过摄像头检测圆形（球），利用颜色阈值识别球的颜色。

根据目标颜色（红/蓝）和多帧缓冲决策，控制机器人旋转、前进/后退，逐步靠近目标球。

当球在画面中的偏移量和大小满足靠近条件时，停止移动并返回 "stop"，表示抓球成功。

deliver_ball.py

功能：将球送到对应颜色的安全区（紫色目标区域）

检测紫色色块（代表球门/目标位置），并识别红/蓝色安全区。

控制机器人旋转或平移，使紫色区域居中，同时确保目标颜色的安全区出现（若出现其他颜色则原地转向）。

当紫色区域居中且与安全区匹配时，返回 "stopstop"，表示送球完成。

find_and_grab_ballqq.py

功能：抓取后续球（优先大球，否则找最近小球）

优先检测大球（半径 8~25 像素），按颜色优先级（黑 > 目标色）选择目标，并控制机器人靠近。

如果连续多帧未检测到大球，则改为检测所有圆形（小球），寻找最近的目标颜色球靠近。

成功靠近后返回 "stop"，表示本次抓球完成。
